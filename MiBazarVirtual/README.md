# MiBazarVirtual

Marketplace de alimentos desarrollado como proyecto de grado de Ingeniería en Sistemas.

## Estructura

- `backend/`: API Spring Boot, autenticación, productos, pedidos, chat y configuración.
- `frontend/`: aplicación móvil React Native con Expo.
- `database/`: scripts SQL de esquema, datos semilla e índices.
- `docs/`: documentación técnica, diagramas y entregables del proyecto.
